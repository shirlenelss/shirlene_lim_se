# shirlene_lim_se
This is my hugo project for my personal blog, where I write about DevSecOps, infrastructure, 
and all things pipelines. The content is in the `content` directory, 
and the theme is a custom one based on the popular "m10c" theme.

The site is built with Hugo, a static site generator that allows me to write in Markdown and 
generate a fast, secure, and easily deployable website.
